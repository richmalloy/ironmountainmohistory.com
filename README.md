# Iron Mountain, Missouri — A Documentary History

Public history site for the Iron Mountain mine property, St. François County, Missouri (T35N R4E). Live at [ironmountainmohistory.com](https://ironmountainmohistory.com).

## Methodology

Every claim on the site is tagged with its source and verification status:

- **Verified** — traced to a named primary source (period newspapers, military records, state mine maps)
- **Documented** — supported by secondary sources, identified as such
- **In progress** — an open research thread, presented as a question rather than a fact

Corrections and leads welcome via issues.

## Known conflation traps

Flagged so researchers don't repeat them:

- "Camp Iron Mountain" is a WWII-era facility in **California**, unrelated to this property.
- The Missouri Mines State Historic Site in Park Hills is an unrelated **lead**-mining site.
